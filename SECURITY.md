# Security Policy

## Supported Versions

We actively provide security updates for the following versions of `mcp-prompt-optimizer-local`:

| Version | Supported          |
| ------- | ------------------ |
| 4.0.x   | :white_check_mark: |
| 3.1.x   | :x:                |
| < 3.1   | :x:                |

## Reporting a Vulnerability

We prioritize the security and privacy of our users. Since the Local Edition handles prompts on-device, security is paramount. If you discover a security vulnerability, please do NOT open a public issue. Instead:

1.  **Email us**: Send a report to support@promptoptimizer.xyz.
2.  **Include details**: Describe the vulnerability, steps to reproduce, and impact.
3.  **Response time**: We aim to respond within 48 hours.
4.  **Disclosure**: We will coordinate public disclosure once a patch is ready.

Thank you for helping keep the Local Edition secure!
